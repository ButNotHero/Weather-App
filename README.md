# Прогноз погоды

Данный проект является тестовым заданием.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints files
```
yarn lint
yarn lint:eslint
yarn lint:prettier
yarn lint:stylelint
```

### Fixes files
```
yarn fix
yarn fix:eslint
yarn fix:prettier
yarn fix:stylelint
```
